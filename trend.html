6  Warehouse area trend

The area covered by warehouses in Southern California has increased over time.

This analysis shows the trend in warehouses since 1980 and the projected time to complete the backlog of warehouse projects in various stages of pre-development.

  • Transform and visualize data to estimate trend in warehouse growth
# create a new data table 
wh_area_by_year <- warehouses |> 
  group_by(year_built) |> 
  summarize(footprint = sum(shape_area),
            count = n(), .groups = 'drop') |>
  filter(!is.na(year_built)) |> 
  arrange(year_built) |> 
  mutate(acres = round(footprint/43560, 1)) 

wh_area_summed_by_year <- wh_area_by_year |> 
  reframe(sumAcres = cumsum(acres)) |> 
  bind_cols(wh_area_by_year)

##calculate slope of trend
lm(sumAcres~year_built, data = filter(wh_area_summed_by_year, year_built %in% c(1981:2025)))

    year_built %in% c(1981:2025)))

# display area built from 1981-2025
ggplot(filter(wh_area_summed_by_year, year_built %in% c(1981:2025)), 
       aes(x = year_built, y = sumAcres)) + 
  geom_col() +
  theme_bw() +
  labs(title = 'Warehouse area in Southern California (1981-2025)',

6.2 Estimate the time to build out the backlog of approved and in-process warehouse projects

Use the slope, intercept, and warehouse area for future years to estimate the backlog of warehouse projects available for construction in Southern California.

#
approvedArea <- wh_area_summed_by_year |> 
  filter(year_built == 2027) |> 
  pull(sumAcres)
CEQAReviewArea <- wh_area_summed_by_year |> 
  filter(year_built == 2030) |> 
  pull(sumAcres)

## y = mx + b
## x = (y-b)/m
# b = -2151926
# m = 1092

est_year_approved <- round((approvedArea + 2151926)/1092, 0)
CEQARevew_year_approved <- round((CEQAReviewArea + 2151926)/1092, 0)
  • Update dataset and visualize buildout years for warehouse growth
trend_data <- wh_area_summed_by_year |> 
  mutate(year_built = case_when(
    year_built == 2027 ~ est_year_approved,
    year_built == 2030 ~ CEQARevew_year_approved,
    .default = year_built
  ))

# display area built from 1981-2025
ggplot(trend_data, 
       aes(x = year_built, y = sumAcres)) + 
  geom_col(width = 0.8) +
  theme_bw() +
  labs(title = 'Warehouse area in Southern California (projected buildout)',
       x = 'Year',
       y = 'Acres',
       caption = 'Data from Warehouse City v1.24') +
  geom_smooth(method = 'lm', color = 'orange', se = T) +
  annotate(geom = 'text', x = 2037, y = 60000, label = 'Approved') +
  annotate(geom = 'text', x = 2045, y = 90000, label = 'Under Review') +
  scale_x_continuous(breaks = c(1980, 1990, 2000, 2010, 2020, 2030, 2040, 2050)) +
  scale_y_continuous(breaks = c(0, 20000, 40000, 60000, 80000))

This plot shows that at the average rate of buildout from 1980-2025, the approved warehouse projects in Southern California would be built out by 2042, or 16 years from now. The under review warehouse backlog runs through 2053.

Southern California industrial real estate for warehouse uses is prepared for over 25 years of future growth unless construction rates increase substantially over the long-term trend.
